**Connect** is a modern social media web application frontend built with **React**.  
It allows users to register, log in, create posts, like/unlike posts, comment, manage friends, and chat in real time with a clean and responsive UI.

## Features

- User Authentication (Register & Login)
- Form handling & validation using **Formik** and **Yup**
- Create posts with images
- Like / Unlike posts
- Comment on posts
- Add / Remove friends
- Real-time chat using **Socket.io**
- Profile picture & post image uploads (Cloudinary)
- Persistent login state using Redux Persist
- Fully responsive UI with **Material UI**
